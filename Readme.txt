
есть две шапки: одна имеет id="header-fix"
Внутри такой шапки на больших экранах меню категорий изначально 
отображается в качестве левого сайдбара и имеет класс catalog-menu--visible. 
Этот id у шапки нужен для скрипта, который работает с этим меню категорий

Если меню изначально не отображается, то у шапки нет этого id и у каталога нет класса catalog-menu--visible .


******************  Главная   ****************************

Главная 1 отличается от Главной 2 только слайдером в шапке. 
Первый вариант - просто слайдер
Второй вариант - сложная сетка  на первом экране




******************  Страница товара  ****************************

Страница товара 2 имеет дополнительный переключатель в табах Характеристики. А так же в описании товара есть две ссылки:
первая прокручивает к контенту в табе
Вторая открывает вкладку Характеристики

******************  Страница категорий  ****************************

Карточки на странице категорий могут переключаться в три вида.  Скрипт описан в файле display-toggle.js

1) Кнопка с data-display="row"  отображает крупные горизонтальные  карточки по одной в ряду
   Это построение вызвано наличием класса третьего класса  "wrapper-column"   у блока "category-products filtered-block-content"

   Сама кнопка имеет вид активной за счет класса active  "btn-display-row active"


2 Кнопка с data-display="grid" отображает   карточки по 4 в ряду
   Это построение вызвано наличием только двух классов   "category-products filtered-block-content"

   Сама кнопка имеет вид активной за счет класса active  "btn-display-grid active"

2 Кнопка с data-display="table" отображает   карточки в виде узких рядов таблицы
   Это построение вызвано наличием четырех классов  "category-products filtered-block-content wrapper-column wrapper-table"

   Сама кнопка имеет вид активной за счет класса active  "btn-display-table active"

Если надо по загрузке страницы отобразить один из трех видов, то надо добасит блоку с карточками соотвествующие классы

******************  Страница одной категорий  ****************************

Вторая, когда в категории нет товаров
